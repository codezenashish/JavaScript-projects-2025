## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic text editor or IDE
- Local web server (optional, for some projects)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/codezenashish/JavaScript-projects-2025.git
   ```

2. **Navigate to the project folder**

   ```bash
   cd JavaScript-projects-2025
   ```

3. **Open any project**
   ```bash
   cd 01-qr-code
   # Open index.html in your browser
   ```

### Running Projects

Most projects can be run by simply opening the `index.html` file in your browser. For projects with dependencies:

```bash
# For projects with npm packages (e.g., 02-signup-page)
npm install
npm run build
```
